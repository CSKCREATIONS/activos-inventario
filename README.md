# activos-inventario
 para iniciar el proyecto se debe ejecutar el comando `npm install` para instalar las dependencias necesarias. Luego, se puede iniciar el servidor con el comando `npm start`. El proyecto utiliza React para la interfaz de usuario y Node.js para el backend. Asegúrate de tener ambos entornos configurados correctamente para un funcionamiento óptimo.

 para encender de manera local el proyecto se debe ejecutar el comando `npm start` en la terminal. Esto iniciará el servidor de desarrollo y abrirá la aplicación en tu navegador predeterminado. Asegúrate de tener todas las dependencias instaladas y configuradas correctamente antes de ejecutar este comando.

 para encender el backend de manera local, navega a la carpeta `backend_py` y ejecuta el comando `python main.py` o `uvicorn main:app --host 0.0.0.0 --port 8000` para ejecutarlo en modo desarrollo.
